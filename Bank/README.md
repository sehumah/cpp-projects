# Bank

Bank is a C++ console project that simulates banking operations:

- Opening accounts
- Deposits
- Withdrawals
- Account-to-Account Money Transfers

More features will be implemented regularly.

Compile and execute the application with:

```bash
g++ -std=c++20 {Account,Bank,main}.cpp -o a.out

./a.out
```

## Version History

__Version 1__ has the following features:

- opening accounts
- depositing money
- withdrawing money
- transfering money
- viewing all accounts
